# Twicuss Train Mod

連結可能、より直感的に運転ができるトロッコを追加します。

連結方法
-
レンチを手にもち連結したい二つのトロッコを順にシフト右クリック

レシピ
-
- レンチ  
  
　　□XX  
　　□XX  
　　X□□
  
　X: 鉄インゴット  
  
- トロッコ  
  
　　X    
　　Y  
  
　X: レッドストーン  
　Y: トロッコ  
  
- チェスト付きトロッコ  
  
　　X  
　　Y  
  
　X: チェスト  
　Y: 上のトロッコ  
  
- 点字ブロック  
  
　　XXX
  
　X: 黄色のカーペット
